## ההבדל בין אוטומציה רגילה לאוטומציה חכמה

אוטומציה רגילה: "אם השדה 'סטטוס' שווה 'חדש' — שלח מייל."

אוטומציה עם AI: "קרא את המייל שהגיע. אם הלקוח נשמע דחוף — תייג אותו 'עדיפות גבוהה' ושלח הודעה ל-WhatsApp. אם הוא שואל על מחירים — שלח לו את ה-PDF. אם זה ספאם — מחק."

ההבדל הוא **מודול AI** בתוך ה-Scenario של Make. הוא קורא טקסט ומחזיר החלטה — ושאר ה-Scenario מגיב בהתאם.

## הצעדים

<Step number={1}>
ב-Make, צרו Scenario חדש (לחצו על **"Create a new scenario"**). הגדירו את ה-Trigger — למשל Gmail: "Watch Emails" כדי לתפוס מיילים נכנסים.
</Step>


<Step number={2}>
הוסיפו מודול חדש — לחצו על **+** אחרי ה-Trigger וחפשו **"Claude AI"** או **"OpenAI"**. בחרו את השירות שיש לכם API Key אליו.

למבתחילים — **Claude AI** (ב-Make) קל יותר להגדרה.
</Step>


<Step number={3}>
הגדירו את ה-Prompt של ה-AI. זה הלב של האוטומציה. לדוגמה:

```
אתה עוזר לסווג מיילים נכנסים לעסק. קרא את המייל הבא והחזר JSON בלבד:

נושא המייל: {{1.subject}}
גוף המייל: {{1.snippet}}

החזר:
{
  "category": "lead" / "support" / "spam" / "vendor",
  "urgency": "high" / "medium" / "low",
  "summary": "סיכום קצר של 1 משפט",
  "suggested_action": "מה לעשות עם המייל הזה"
}
```

שימו לב ל-`{{1.subject}}` — Make ממלא שם אוטומטית את ערכי המייל.
</Step>


<Step number={4}>
אחרי מודול ה-AI, הוסיפו **Router** — מחלק שמפנה לנתיבים שונים לפי התשובה:

- `category = "lead"` → הוסיפו ל-CRM + שלחו מייל מעקב
- `urgency = "high"` → שלחו הודעה ב-Slack / WhatsApp
- `category = "spam"` → מחקו אוטומטית

ב-Make מגדירים זאת ב-**Filters** על כל נתיב של ה-Router.
</Step>


<Tip>
בקשו מה-AI תמיד להחזיר **JSON בלבד** — זה מקל על ה-Router לקרוא את התשובה. הוסיפו בסוף ה-Prompt: "אל תוסיפו הסברים — JSON בלבד."
</Tip>

## שימושים עסקיים אמיתיים

**סיווג לידים נכנסים:**
כל ליד שמגיע מהאתר → AI קורא את ה"הערות" שמילא → מסווג לפי חום הליד → מפנה לנציג המתאים ב-CRM.

**עיבוד בקשות תמיכה:**
לקוחות שולחים מייל → AI מבין את הבעיה → מחזיר קטגוריה + דחיפות → המיילים נמשכים לתיקייה הנכונה ו-Slack נשלח לאחראי.

**ניתוח פידבקים:**
אחרי כל פגישה, סורקים את הסיכום → AI מזהה נקודות כאב, החלטות, ו-Action Items → מעדכן את ה-CRM עם insights.

## מה צריך להתחיל?
